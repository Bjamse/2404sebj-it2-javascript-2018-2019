popp opp bokser

alert()

Alert er en popp opp boks med bare text.
texten sendes som et parameter til alert funksjonen.
De kan skrives slik:

alert("Noe text");

confirm()

akkuratt lik alert, bare at brukeren får en godkjenn og en avbryt knapp
Disse kan brukes til å presenter boolske avgjørelser for brukeren
Her er et eksempel :

// deklarerer global variabel for om nettsiden er rotert eller ikke akkuratt nå
let rotert = false;

// funksjon for å rotere nettsiden
function rot(){
  // sjekk om brukeren vil rotere skjermen eller ikke
  if(confirm("Roter siden?")){
    //roter skjermen, hvis den er rotert så roter tilbake;
    document.getElementById("wrapper").style.transform = rotert? "rotate(180deg)":"rotate(0deg)";
    //flipper verdien om siden er rotert eller ikke
    rotert = !rotert;
  }
}


prompt()

prompt er en alert boks med et input felt
den kan brukes slik:

let x = prompt("skriv inn en text som skal vises i ramma under");

document.getElementById("ramme").innerText = x;
